# Business_Python_Basic
## 💡 목표

파이썬을 처음 접하거나 프로그래밍을 처음 배우는 사람들에게 도움을 줄수있는 강의를 만들고자 한다. </br> 
개발의 기초가 될수있게 해주며 웹개발, 데이터 분석, 머신러닝 등등에 필요없는 내용은 뺴고 </br> 
최대한 개발에 실무적인 도움이 될수있는 부분에 한하여 정리하고자 한다.
